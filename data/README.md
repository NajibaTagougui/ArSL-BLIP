# Data Directory

This directory contains preprocessing scripts and configuration for the ArASL dataset.

## Dataset

The model uses the **ArASL (Arabic Alphabets Sign Language)** dataset:

- **32 classes**: All Arabic alphabet letters
- **54,049 images** (original), **108,098** (after augmentation)
- **200 images per class** sampled for our experiments
- Available at: [Mendeley Data](https://data.mendeley.com/)

## Structure

```
data/
├── README.md
├── raw/              ← Place original ArASL dataset here (not tracked by git)
├── processed/        ← Preprocessed splits (generated by scripts)
└── preprocessing/
    ├── preprocess.py       ← Image preprocessing pipeline
    ├── dataset_split.py    ← Train/val splitting
    └── augment.py          ← Data augmentation utilities
```

## Setup

```bash
# 1. Download ArASL dataset and place in data/raw/
# 2. Run preprocessing
python data/preprocessing/preprocess.py --source data/raw --output data/processed

# 3. Create splits
python data/preprocessing/dataset_split.py --data data/processed --train 0.85 --val 0.15
```
